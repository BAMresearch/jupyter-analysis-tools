jupyter_analysis_tools.notebook_utils.currentNBpath#

jupyter_analysis_tools.notebook_utils.currentNBpath()[source]#

Returns the absolute path of the Notebook or None if it cannot be determined NOTE: works only for Jupyter Notebook (not Jupyter Lab) and when the security is token-based or there is also no password.
